### Retail Digital Store
Retail digital store is a type of application that establishes discount calculation according to the given constraints.
Retail digital store is built with the following: Java, Junit, ArgoUML and Maven tool.

#Feature
- Discount calculation
- TDD
- DDD
- UML Diagram

#Installation & Usages
- STEP 1: Clone repository
- STEP 2: Install dependencies: cd into root folder run the following command `mvn install` then `mvn compile`
- STEP 3: Run the project `java -cp target/DigitalSupermarket-1.0-SNAPSHOT.jar com.store.digital.supermarket.RetailStoreDiscountApp` to test the JAR compiled file.
- STEP 3: The above command can also work with the test cases.
